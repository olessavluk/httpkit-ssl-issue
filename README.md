# httpkit-ssl-issue

## Usage

In order to run this example with `lein`, run:

    $ lein run
    $ lein run https://google.com https://clojure.org
    
or using docker

    $ docker run olessavluk/httpkit-ssl-issue
    $ docker run olessavluk/httpkit-ssl-issue https://google.com https://clojure.org
